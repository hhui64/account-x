ACCOUNT-X 是一个使用 NestJS 框架开发的基于 Yggdrasil 服务端技术规范的 Minecraft 用户管理系统。

## 功能
### 基本功能
- 邮箱注册/登录
- 多角色管理
- 皮肤上传及预览
- 后台管理面板
- 更多功能...

### 内在特性
- 支持 Redis 缓存
- 多种 API 接口供插件调用
- 更多特性...
